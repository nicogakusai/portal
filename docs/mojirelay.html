文字リレー企画

当企画「文字リレー企画」は集まって下さった参加者の皆様で、それぞれ起承転結を分担して貰い
一つの文章作品を作り上げます。

一緒に作り上げた物語でみんなを笑顔にしてみませんか？

文章作品未経験者も大歓迎。
普段、アレンジ、イラスト、MMD等の創作をされている方も大歓迎です。
新しい創作の世界に足を踏み入れてみませんか？

バックストーリー

紅魔館の主は暇を持て余していた。
ふと気付くとテーブルには一通の手紙が置かれている。
目線を挙げるとそこには瀟洒なメイドが微笑んでいた。
手紙の差出人は本居小鈴。
いつかペットが迷惑をかけた時があった。
その時に書簡を送ったのだが、以来時折やりとりをする仲になった。
なんでも彼女は妖魔本コレクターで妖怪が書いた物なら手紙ですらコレクションしているらしい。
やりとりを始めた頃、嬉しい嬉しいと何度も書かれていたっけ・・・

そうだ！
図書館に足を向け最近お荷物呼ばわりされている親友に声をかける。
突飛な考えに最初は難色を示していた親友も最終的には根負けしてしまった。
メイドに声をかけお出かけの準備をする。次は何処にしよう？

小鈴は色々な妖怪の手紙をほしがっている。
なら妖怪の連名書簡はきっと喜ぶはずだ。
ただ手紙を書くだけでは芸がない。
なら連名で一つの芸術、一つの妖魔本を作ってみようじゃないか！

参加に当たっての注意点

  • 当企画はお笑い企画です。紅魔館が爆発しても、登場人物が不幸になっても笑って許せる方のみご参加下さい。
  • 出来上がった作品はニコニコ生放送で朗読形式で放送される予定です。

    • 参加方法

    1. 参加登録フォームより参加情報のご登録をお願い致します。
    2. 参加者募集〆切後、チャットツール「discord」の本企画用サーバのアクセス用URLをお送り致します。
    3. お送りしたURLより、discordサーバへの参加をお願い致します。
    4. 応募期間終了後に、ご担当いただく章をdiscordサーバ内にてお伝え致します。
    5. 作品の制作をお願い致します。
    6. 作品の完成時には、専用の作品提出フォームより作品のご提出をお願い致します。

    スケジュール等

    3月1日～3月31日 参加者募集期間
    4月初め discord本企画用サーバのアクセス用URL送付
