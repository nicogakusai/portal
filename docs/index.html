東方ニコ楽祭は、２０１５年に開催された『東方ニコニコ樂祭』と、お馴染みの『東方ニコ童祭』をリスペクトしています。


第15回東方ニコ童祭 生放送番組「第四回幻想コメント大賞」

皆様からご投稿頂いたイラストと、そのイラストを見て思いついたおもしろコメントをご投稿いただく、大喜利企画です。
第15回東方ニコ童祭 開催期間中の生放送で皆様と鑑賞しつつ、一番おもしろい回答「幻想コメント大賞」を決定します！

イラスト参加

イラストテーマ：四季（春夏秋冬）　※このテーマにあったイラストを作成してください。
イラスト投稿期間：7/5(水)24時まで
投稿先：ニコニコ静画
タグ：「第四回幻想コメント大賞」、「東方動画イラスト支援」の2つをロックする
注意事項：

  • 1人につき1枚までの投稿とします。
  • R18/R18-Gの作品はご遠慮ください。
  • 台詞吹き出しはなしとします。
  • 生放送、動画等で紹介される場合があります。

    • コメント参加

    本企画の参加イラストのコメント欄でおもしろ回答をコメントしてください。
    コメント投稿期間：7/6(木)21時まで
    注意事項：

  • 1人あたりの回答数に制限はありません。
  • コメントの後ろに「@お名前」をつけてコメントしてください。
    ※お名前がない場合は、「匿名投稿者」として紹介されます。
  • 生放送では、全てのコメントが紹介できない可能性があります。予めご了承ください。
